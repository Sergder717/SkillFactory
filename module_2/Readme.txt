В описании (README) кратко укажите:
I. Основные цели и задачи проекта:
   - проверка усвоения пройденной темы;
   - разобр на реальном кейсе, с какими задачами придется сталкиваться в будущем;
   - научиться понимать трактование заданий и проводить анализ для получения корректного ответа;
   - научиться находить способы решения самостоятельно;
   - получить базовые знания и навыки по проведению EDA, научиться анализировать информацию, находить связи;
   - научиться самостоятельно работать с документацией python и др. для подбора функций для более качественного решения;
   - научиться пользоваться slack для консультаций по возникшим вопросам с менторами.

II. Краткая информация о данных:
   - В данных было много пропусков, в количественных переменных значения были заменены на медиану, в номинативных на моду;
   - Выбросы были найдены в переменных: students_age, father_education, absences, family_relationships;
   - Переменные "studytime_granular" с "study_time" скоррелированы почти полностью обратной корреляцией. 
     "studytime_granular" был исключен из анализа;
   - Положительная корреляция параметра mother_education и father_education может говорить о 
     положительном влиянии на рост успеваемости учеников образования родителей, и довольно странная связь со столбцом absences говорит о том, что отсутствие пропусков занятий плохо сказывается на успеваемости.
   - Из номинативных переменных оставим в датасете следующие переменные: 
     sex, address, additional_paid_lessons, get_higher_education, in_romantic_relationship
   - Самые важные параметры, которые предлагается использовать в дальнейшем для построения модели, это:
     absences, failures, mother_education, father_education, go_out, health_status, sex, address, 
     additional_paid_lessons, get_higher_education, in_romantic_relationship
