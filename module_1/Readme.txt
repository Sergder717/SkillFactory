В описании (README) кратко укажите:
I. Задача проекта:
   - проверка усвоения пройденной темы;
   - разобр на реальном кейсе, с какими задачами придется сталкиваться в будущем;
   - научиться понимать трактование заданий и проводить анализ для получения корректного ответа;
   - научиться находить способы решения самостоятельно;
   - получить базовые знания по pandas, датафреймами и подготовке данных для анализа;
   - научиться самостоятельно работать с документацией python и др. для подбора функций для более качественного решения;
   - научиться пользоваться slack для консультаций по возникшим вопросам с менторами.

II. Количество набранных в викторине баллов: 27/27

III. Ответы на вопросы саморефлексии:

   1. Какой частью своей работы вы остались особенно довольны?
      - Решение всех задач и получение 27 балов из 27.
      - Способность самостоятельно находить решение задач
      - В какой-то степени достижение всех результатов из п.1.   

   2. Что не получилось сделать так, как хотелось? Над чем ещё стоит поработать?

        хотелось бы вернуться к разбору задач после комментариев ментора либо вебинара.
	
   3. Что интересного и полезного вы узнали в этом модуле?
      - Прменение функций для редактирования столбцов или создания новых столбцов;

      - Научиться работать с датафреймами;

      - Практика в написании качественного и оптимального кода;
      - Разбор слабых мест проекта, после комментариев ментора. 

   5. Планируете ли вы менять стратегию изучения последующих модулей?

        но не всегда получается уделять времени больше, чем изучение только основных модулей.
